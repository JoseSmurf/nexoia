# Changelog

## [1.0.0] - 2026-06-13

STABLE: nex v1.0.0 is FROZEN.

- Freezes the nex language contract at `1.0.0` with grammar version `1`.
- Stabilizes the 7 constructs: version header, `use`, `node`, `derive`, `attest`, `assert`, and `act`.
- Stabilizes the type system with `i64` and `string` values.
- Stabilizes the witness model for signed-to-anchored attestation.
- Stabilizes the import mechanism through dot paths resolved to `.nex` files.
- Stabilizes `program_hash` as the canonical hash of the expanded program.
